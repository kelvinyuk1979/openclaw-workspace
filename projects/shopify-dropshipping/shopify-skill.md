# Shopify 选品 + 运营技能

**用途：** 用 OpenClaw 自动化 Shopify 跨境店铺运营

---

## 🛠️ 核心任务

### 1. 每日选品调研

**提示词：**
```
像跨境电商专家一样帮我选品。

要求：
1. 搜索过去 7 天内 trending products 2026
2. 聚焦品类：[家居/宠物/美妆/运动/3C]
3. 筛选标准：
   - 售价 $20-50
   - 重量轻（<500g）
   - 无尺码/少 SKU
   - 非季节性
   - Amazon 评论<500
4. 输出 5 个潜力产品，每个包括：
   - 产品名称和描述
   - 目标客户
   - 预估售价
   - 1688/Alibaba 采购价（估算）
   - 利润率
   - 竞争分析（主要卖家、评论数）
   - 推荐理由

用表格形式输出。
```

---

### 2. 竞品分析

**提示词：**
```
帮我分析这个竞品的 Shopify/Amazon 店铺：[店铺链接/ASIN]

分析维度：
1. 产品列表 - 卖什么、价格区间
2. 销量估算 - 评论数、上架时间
3. 评论分析 - 好评点、差评点（找机会）
4. Listing 质量 - 标题、图片、描述
5. 流量来源 - 估算（社交媒体、广告、自然搜索）
6. 弱点分析 - 他们做得不好的地方
7. 机会点 - 我可以如何差异化

输出结构化报告，突出可执行的洞察。
```

---

### 3. Listing 优化

**提示词：**
```
帮我优化这个产品的 Listing：

产品信息：
- 名称：[产品名]
- 功能：[核心功能 1,2,3]
- 目标客户：[谁买]
- 竞品参考：[竞品链接]

生成以下内容：
1. 标题（60-80 字符，SEO 优化，包含核心关键词）
2. 产品描述（300-500 词，突出卖点 + 场景 + 情感）
3. 5 个 Bullet Points（每点 1 个核心卖点）
4. 关键词列表（15-20 个，按搜索量排序）
5. 常见问题 FAQ（5-8 个，提前回答客户疑虑）

语气：[专业/友好/幽默]
目标市场：[美国/欧洲/英国]
```

---

### 4. 客服自动回复模板

**提示词：**
```
帮我生成 Shopify 店铺客服自动回复模板。

常见场景：
1. 发货时间咨询
2. 物流追踪
3. 退换货政策
4. 产品规格询问
5. 订单修改/取消
6. 破损/质量问题
7. 优惠码咨询

每个模板包括：
- 客户问题示例
- 自动回复（英文，友好专业）
- 何时转人工

语气：友好、专业、有同理心
品牌调性：[高端/亲民/年轻]
```

---

### 5. 供应商询价

**提示词：**
```
帮我写供应商询价邮件/消息（中英文版本）。

产品信息：
- 品类：[产品名]
- 目标采购价：$[X]/件
- 首单数量：[X] 件
- 目标市场：[美国/欧洲]

询问内容：
1. 单价（不同数量阶梯）
2. MOQ（最小起订量）
3. 样品政策和费用
4. 生产周期
5. 发货方式和运费
6. 是否支持一件代发
7. 是否有欧美认证（CE/FCC 等）
8. 定制可能性（LOGO、包装）

语气：专业、真诚、长期合作意向
```

---

### 6. 利润测算

**提示词：**
```
帮我测算这个产品的利润：

输入：
- 采购价：$[X]/件
- 售价：$[X]/件
- 运费：$[X]/件
- 平台佣金：[X]%
- 支付手续费：[X]%
- 广告成本：$[X]/件（估算）

计算：
1. 毛利润 = 售价 - 采购价 - 运费
2. 净利润 = 毛利润 - 佣金 - 手续费 - 广告
3. 利润率 = 净利润/售价

输出：
- 利润测算表
- 盈亏平衡点（需要卖多少件）
- 敏感性分析（售价/成本变动对利润影响）
- 建议（是否值得做）
```
