## 📧 战报内容

### 邮件主题
```
⚔️ A 股量化战报 - YYYY-MM-DD | 盈亏：+/-¥XXX.XX (+/-X.XX%)
```

### 邮件内容
| 模块 | 内容 |
|------|------|
| 📊 **今日盈亏** | 盈亏金额 + 收益率（大字显示） |
| 💰 **账户概览** | 账户总值 / 可用现金 / 持仓数量 |
| 📈 **持仓详情** | 股票代码 / 数量 / 成本价 / 盈亏% |
| 📝 **今日交易** | 时间 / 操作 / 股票 / 价格 / 数量 / 盈亏 |
| 📊 **本周累计** | 交易日 / 收益率 / 交易数 / 胜率 |

## 📅 每日流程

```
09:30  开盘 → 运行量化策略 → 扫描市场 → 生成信号
        ↓
        执行交易 → 更新持仓 → 记录数据
        ↓
15:00  收盘 → 计算盈亏 → 生成报告
        ↓
15:30  发送战报邮件 → 你的邮箱
        ↓
        你收到今日战况 📧
```
