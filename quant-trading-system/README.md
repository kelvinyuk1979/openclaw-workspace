# 🤖 多市场量化交易系统

**版本：** 1.0.0  
**创建日期：** 2026-03-16  
**模式：** 模拟交易 → 全自动

---

## 📋 系统概述

这是一个支持**A 股、港股、Crypto**多市场的量化交易系统，采用**4 策略组合**，实现自动化交易。

### 核心特性

| 特性 | 说明 |
|------|------|
| 🌍 **多市场支持** | A 股 + 港股 + Crypto，24 小时交易 |
| 📈 **4 策略组合** | 趋势跟踪 + 动量 + 均值回归 + 多因子 |
| 🔒 **严格风控** | 止损/止盈/仓位/熔断机制 |
| 📊 **日线频率** | 每日收盘前调仓，适合上班族 |
| 💰 **模拟先行** | 先模拟验证，再考虑实盘 |

## 🎯 交易策略

### 策略配置

| 策略 | 资金分配 | 预期年化 | 最大回撤 | 核心逻辑 |
|------|----------|----------|----------|----------|
| 📈 **趋势跟踪** | 30% | 20-30% | -15% | MA 金叉/死叉 + MACD + RSI |
| 🚀 **动量策略** | 30% | 25-40% | -20% | 突破 20 日高点 + 成交量确认 |
| 📊 **均值回归** | 25% | 15-25% | -12% | 布林带 + RSI 超买超卖 |
| 💎 **多因子选股** | 15% | 18-28% | -15% | 价值 + 成长 + 质量因子 |

### 策略详情

#### 1. 趋势跟踪策略 (Trend Following)
**核心指标：**
- MA5 / MA20 金叉死叉
- MACD 金叉死叉
- RSI 过滤超买超卖

**买入信号：**
- ✅ MA5 上穿 MA20
- ✅ MACD 金叉
- ✅ RSI < 70

**卖出信号：**
- 📉 MA5 下穿 MA20
- 📉 MACD 死叉

---

#### 2. 动量策略 (Momentum)
**核心指标：**
- 20 日价格动量
- 成交量比率
- 布林带突破

**买入信号：**
- ✅ 突破 20 日高点
- ✅ 成交量>1.5 倍均值
- ✅ 动量为正

**卖出信号：**
- 📉 跌破 20 日低点
- 📉 动量转负

---

#### 3. 均值回归策略 (Mean Reversion)
**核心指标：**
- 布林带上下轨
- RSI 超买超卖

**买入信号：**
- ✅ 触及布林带下轨
- ✅ RSI < 25 超卖

**卖出信号：**
- 📉 触及布林带上轨
- 📉 RSI > 75 超买

---

#### 4. 多因子选股策略 (Multi-Factor)
**核心因子：**
- 价值因子（PE 倒数等）30%
- 成长因子（收入/利润增长）30%
- 质量因子（ROE/ROIC）25%
- 动量因子 15%

**买入信号：**
- ✅ 综合得分>70
- ✅ 各因子>60

**卖出信号：**
- 📉 综合得分<40

## 🔒 风险控制

### 仓位管理

| 风控项 | 设置 | 说明 |
|--------|------|------|
| 单只股票最大仓位 | 10% | $1,000 / 只 |
| 总仓位上限 | 80% | $8,000 最大持仓 |
| 现金保留 | ≥20% | $2,000 备用金 |

### 止损止盈

| 类型 | 触发条件 | 操作 |
|------|----------|------|
| 止损 | -8% | 自动卖出 |
| 止盈 | +25% | 卖出 50% |

### 熔断机制

| 指标 | 限制 | 触发操作 |
|------|------|----------|
| 单日最大亏损 | -5% ($500) | 停止当日交易 |
| 单周最大亏损 | -10% ($1,000) | 减仓 50% |
| 单月最大亏损 | -15% ($1,500) | 清仓观望 |
| 策略连续亏损 | 3 天 | 暂停该策略 |

## 🔧 配置说明

### config.json 关键配置

```json
{
  "account": {
    "initial_capital": 10000,    // 初始资金
    "mode": "simulation"         // simulation / live
  },
  "risk_control": {
    "stop_loss": -0.08,          // 止损 -8%
    "take_profit": 0.25,         // 止盈 +25%
    "position_single_max": 0.10  // 单只最大 10%
  },
  "execution": {
    "auto_execute": false,       // 模拟阶段设为 false
    "require_confirmation": true // 需要确认
  }
}
```

### 切换到实盘交易

**⚠️ 仅在模拟验证有效后考虑！**

1. 修改 `config.json`：
```json
{
  "account": {"mode": "live"},
  "execution": {"auto_execute": true}
}
```

2. 配置券商 API 密钥
3. 小额测试（建议<$1000）
4. 逐步增加资金
