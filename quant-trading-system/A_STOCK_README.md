## 🎯 A 股特色功能

### 1. T+1 交易规则
```
✅ 今日买入，明日才能卖出
✅ 自动记录今日买入股票
✅ 止损不受 T+1 限制（紧急情况下）
```

### 2. 涨跌停限制
```
✅ 主板 ±10%
✅ 创业板/科创板 ±20%
✅ 涨跌停时自动跳过交易
```

### 3. 最小交易单位
```
✅ 最小 100 股（1 手）
✅ 自动计算符合规则的股数
✅ 避免零股交易
```

### 4. A 股费用结构
