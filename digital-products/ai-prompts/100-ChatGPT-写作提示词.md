# 🎯 100 个 ChatGPT 写作提示词模板

> 让 AI 帮你写出爆款内容，每天节省 2 小时

---

## 📝 使用说明

1. 复制任意提示词到 ChatGPT
2. 替换 `[括号内]` 的内容为你的具体信息
3. 根据需要调整语气和风格
4. 生成后微调即可发布

## 一、社交媒体内容 (1-20)

### 1. 小红书爆款笔记
```
你是一位小红书爆款内容创作者，擅长写 [插入领域] 相关的笔记。

请帮我写一篇关于 [插入主题] 的笔记，要求：
- 标题：用数字 + 痛点/收益 + 情绪词，不超过 20 字
- 正文：分 3-5 个要点，每点有 emoji，口语化
- 标签：10 个相关 hashtag
- 语气：像闺蜜聊天一样亲切

目标读者：[插入人群，如 25-35 岁职场女性]
```

### 2. 朋友圈带货文案
```
你是一位朋友圈带货高手，擅长用生活化场景种草。

请帮我写一条卖 [产品名称] 的朋友圈文案：
- 开场：用个人使用体验引入
- 中间：3 个核心卖点 + 使用场景
- 结尾：限时优惠 + 购买方式
- 配图建议：3 张图的拍摄角度

语气要真实，像朋友推荐，不要硬广。
```

### 3. 公众号文章大纲

## 🎁 Bonus：高级技巧

### 提示词优化公式
```
好提示词 = 角色设定 + 任务描述 + 具体要求 + 输出格式 + 示例参考
```

### 迭代技巧
```
第一轮：生成初稿
第二轮："请让语气更 [插入风格]"
第三轮："请增加 [插入元素] 的例子"
第四轮："请精简到 [插入字数] 字以内"
```

### 组合使用
```
把多个提示词串联起来，例如：
1. 先用提示词 1 生成大纲
2. 再用提示词 3 展开内容
3. 最后用提示词 81 整理格式
```
