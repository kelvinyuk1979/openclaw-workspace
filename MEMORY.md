## 📈 股票分析框架（华尔街风格）

**学习日期：** 2026-03-14

### 核心分析维度

1. **商业模式与收入流** - 公司如何赚钱
2. **竞争优势/护城河** - 品牌、网络效应、切换成本、成本优势、专利
3. **行业趋势** - 市场规模、增速、技术变革
4. **财务健康** - 收入增长、净利润、自由现金流、利润率、债务、ROE
5. **主要风险** - 经济、行业颠覆、竞争、监管、财务风险
6. **估值对比** - PE、DCF、行业平均、低估/高估判断
7. **情景分析** - 乐观/中性/悲观假设
8. **12-24个月前景** - 催化剂、增长潜力

### 选股范围扩展

- ✅ A股（沪深京）
- ✅ 港股（HKEX）
- ✅ 美股（NYSE/NASDAQ）

### 日报格式要求

每日选股日报应包含：
- 核心推荐（1-3只）
- 华尔街风格深度分析
- 多头/空头辩论视角
- 明确的投资建议（买入/持有/回避）
- 风险提示

## 📈 Stock Market Pro 专业股票分析

**学习日期：** 2026-03-14
**来源：** ClawHub (clawhub.ai/kys42/stock-market-pro)

### 支持市场
- A 股、港股、美股、ETF、期货、外汇

### 核心功能
| 模块 | 功能 |
|------|------|
| 📊 实时行情 | 多市场实时数据 |
| 📈 技术分析 | MA/MACD/RSI/KDJ/BOLL，交易信号 |
| 💰 基本面分析 | 估值/增长/盈利/健康度评分 |
| 🎯 量化选股 | 价值/成长/动量/质量因子 |
| 📉 风险评估 | VaR/CVaR/Beta/夏普比率 |
| 📋 报告生成 | 个股深度报告/每日简报 |

### 量化策略
- **价值**: PE<30, PB<5, 股息率>2%
- **成长**: 收入增长>10%, 利润增长>15%, ROE>15%
- **动量**: 1 月>5%, 3 月>10%, 6 月>15%
- **质量**: ROE>15%, ROIC>10%, 负债率<50%

### 高级选股策略（7 大专业策略 + 小盘股整合）

| # | 策略 | 核心逻辑 | 输出 |
|---|------|----------|------|
| 1 | 📉 **低估值选股器** | PE 低于行业平均 + 稳定增长 + 低负债 + 正向 FCF + ROIC 高 + 分析师上调 | 10 家低估公司 + 业务概述/低估原因/风险/内在价值 |
| 2 | 🔍 **内部交易分析** | 多位内部人员 90 天内公开市场买入 + 购买规模有意义 | 前 5 家公司 + 谁买/买多少/历史影响/管理信心 |
| 3 | 🔄 **情绪与现实差距** | 大幅抛售但基本面强劲 + 负面叙事 vs 实际表现 | 5 只情绪与基本面最不一致的股票 |
| 4 | 💰 **股息贵族 ROI** | 连续 25 年增息 + 10 年总回报（股息再投资）+ 派息比可持续 | 按收入可靠性和总回报排名 |
| 5 | 💻 **科技股炒作 vs 基本面** | 收入增长 vs 估值 + 利润率 + FCF + 资本效率 | 3 只高估 + 3 只低估 + 错误定价解释 |
| 6 | 📊 **行业轮动信号** | 利率 + 通胀 + GDP+ 就业 → 经济周期判断 | 未来 6-12 月跑赢/跑输行业 + 逻辑 + 风险 |
| 7 | ⚖️ **风险调整回报优化** | 5 万美金 + 风险偏好（保守/温和/激进）+ 资产配置 | 持仓规模 + 预期回报 + 再平衡规则 + 下行保护 |

## 🎯 机会发现引擎 (Opportunity Discovery Engine)

**学习日期：** 2026-03-20

**目标：** 持续识别能创造财务价值的机会（商业机会、市场低效、技术变革、新兴趋势）

### 分析框架（7 步）

| 步骤 | 核心问题 | 关注点 |
|------|----------|--------|
| 1️⃣ **识别低效** | 哪里存在信息不对称/定价错误？ | 技术颠覆、法规变化、消费者行为转变 |
| 2️⃣ **评估不对称性** | 下行有限、上行巨大？ | 高期望值 > 保证结果 |
| 3️⃣ **及早检测趋势** | 新基础设施/资金流入/采用信号？ | 早期趋势 = 最大机会 |
| 4️⃣ **变现路径** | 实际怎么赚钱？ | 投资、工具/服务、信息套利、自动化、分销 |
| 5️⃣ **竞争格局** | 拥挤度/壁垒/优势？ | 高壁垒 + 低认知 = 宝贵 |
| 6️⃣ **排序机会** | EV/难度/资本/时间？ | 最佳风险/回报比优先 |
| 7️⃣ **创意思考** | 非常规方法/被忽略细分？ | 发掘他人未注意的机会 |

### 应用方向

- 📈 投资标的筛选（股票、加密货币、预测市场）
- 🛠️ 工具/服务构建机会
- 📊 信息套利（跨平台、跨语言、跨时区）
- 🤖 自动化机会（重复流程、人工密集型）
- 🌐 新兴趋势早期布局

**触发时机：** 定期扫描（每周）、重大新闻事件后、用户询问"有什么机会"时
