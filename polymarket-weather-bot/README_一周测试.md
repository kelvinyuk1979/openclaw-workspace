## 📋 测试配置

### 策略参数（已调整）

| 参数 | 原设定 | **新设定** | 调整理由 |
|------|--------|-----------|----------|
| 入场阈值 | 15¢ | **22¢** | 第 1 天数据显示无<15¢信号，22¢可捕获约 33% 信号 |
| 出场阈值 | 45¢ | **50¢** | 给盈利更多空间 |
| 单笔仓位 | 5% | **3%** | 降低风险，允许更多交易次数 |
| 最大交易数 | 5 笔 | 5 笔 | 保持不变 |

### 自动化设置

- ⏰ **每日运行时间**: 8:00 AM (Asia/Shanghai)
- 📧 **邮件报告**: HTML 格式，包含账户概览、交易统计、信号 Top5
- 📂 **数据保存**: `simulation.json` (累计数据) + `today_signals.json` (当日信号)

## ⚠️ 风险提示

1. **数据源差异** - NWS 预报 vs Wunderground 结算可能有偏差
2. **市场效率** - 低价信号可能有其原因（预报不确定性）
3. **流动性** - 模拟交易假设能按市场价格成交，实际可能有滑点
4. **策略风险** - 天气预报本身有误差，胜率可能低于预期

## 🔧 手动操作命令

### 查看当前状态
```bash
cd ~/.openclaw/workspace/polymarket-weather-bot
cat simulation.json
```

### 手动运行一次
```bash
python3 daily_run.py
```

### 查看邮件草稿（如发送失败）
```bash
open email_draft.html
```

### 重置模拟账户
```bash
python3 bot_v1.py --reset
```

### 生成报告
```bash
python3 bot_v1.py --report
```

## 📞 问题排查

### 邮件未收到
1. 检查垃圾邮件箱
2. 查看 `email_draft.html` 是否有内容
3. 检查 QQ 邮箱授权码是否正确

### 机器人运行失败
1. 检查网络连接
2. 查看 `logs/` 目录下的错误日志
3. 手动运行 `python3 bot_v1.py` 查看详细错误

### 数据异常
1. 验证 NWS API 是否正常：访问 https://weather.gov
2. 验证 Polymarket 市场：访问 https://polymarket.com
