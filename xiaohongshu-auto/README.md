# 小红书自动运营系统

## ⚠️ 重要提示

小红书**没有官方公开 API**，自动化发布存在以下限制：

1. **技术限制**: 需要浏览器自动化 (Selenium/Playwright) 或第三方服务
2. **账号风险**: 自动化操作可能触发风控，导致限流或封号
3. **验证码**: 登录和发布可能需要人工验证

## 推荐方案

### 方案 A: 半自动 (推荐) ⭐

- ✅ AI 生成内容 (文案 + 选题 + 标签)
- ✅ 定时提醒发布
- ✅ 人工确认发布 (安全)

### 方案 B: 全自动化 (有风险)

- 使用 Playwright/Selenium 模拟浏览器
- 需要处理登录、验证码、风控
- 建议小号测试

## 快速开始

### 1. 安装依赖

```bash
./run.sh install
# 或手动安装
pip3 install playwright
python3 -m playwright install chromium
```

### 2. 配置账号

```bash
# 编辑 config.json，填写你的账号信息
```

### 3. 生成内容

```bash
./run.sh generate
# 或 python3 content_generator.py
```

### 4. 发布

**全自动发布**:
```bash
./run.sh publish
```

流程:
1. 浏览器自动打开
2. 扫码登录 (首次需要)
3. 自动填写标题、正文、标签
4. 点击发布

**注意**: 首次使用需要人工扫码登录，之后会保存登录状态。

## 内容策略

### 热门选题方向

| 类型 | 说明 | 频率 |
|------|------|------|
| 干货分享 | 知识/技巧/教程 | 高 |

| 产品测评 | 好物推荐/避雷 | 高 |
| 观点看法 | 热点/行业见解 | 中 |

### 文案模板

```
【标题】数字 + 痛点/收益 + 情绪词
例：3 个技巧让我效率提升 10 倍！

【正文】
- 开头：场景/痛点引入
- 中间：3-5 个要点 (分点清晰)
- 结尾：互动/总结

【标签】
#话题 1 #话题 2 #话题 3
```
