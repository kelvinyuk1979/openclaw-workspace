# 🧪 创业提示词测试

**目标：** 用 7 个提示词验证一个创业想法

---

## 🎯 测试案例：AI 视频解读工作室

**背景：**
- 技能：会用 AI 工具（Whisper、ffmpeg、剪映）
- 兴趣：YouTube 内容创作、视频剪辑
- 资源：一台 Mac、OpenClaw、每天 2-3 小时
- 预算：$500

---

## 1️⃣ 想法验证

**提示词：**
```
像经验丰富的创业导师一样分析。评估这个商业想法：
用 AI 工具批量制作"YouTube 视频解读"，发布到 B 站/小红书赚取平台分成和广告费。
评估市场需求、目标受众和竞争对手。
突出最大风险、弱点和机会，并建议实际改进方式。
```

**预期输出：**
- 市场需求：✅ 有人看
- 竞争：⚠️ 已有很多类似账号
- 风险：❌ 版权问题
- 改进：✅ 做"AI 解读"而非搬运

---

## 2️⃣ 细分市场寻找

**提示词：**
```
基于我拥有的技能、兴趣或资源：
- 会用 AI 工具（Whisper、ffmpeg、剪映）
- 有 OpenClaw 自动化框架
- 每天可用 2-3 小时
- 预算$500

生成 10 个可以用很少或无成本启动的现实初创想法。
重点关注市场需求强、目标客户清晰、快速产生收入的想法。
```

**预期输出：**
1. AI 视频解读工作室
2. 跨境电商选品分析服务
3. AI 代写服务（简历、邮件、文案）
4. 自动化交易策略订阅
5. AI 工具培训课
6. ...

---

## 3️⃣ 客户痛点研究

**提示词：**
```
像市场研究员一样思考。针对这个商业想法：
用 AI 工具帮创作者批量制作视频解读内容

找出目标客户面临的主要问题或沮丧。
解释为什么这些问题存在、客户如何尝试解决，
以及愿意为什么样的解决方案付费。
```

**预期输出：**
- 痛点 1：没时间看长视频 → 愿意付费看摘要
- 痛点 2：语言障碍（英文视频）→ 愿意付费看中译
- 痛点 3：找不到优质内容 → 愿意付费看精选

---

## 4️⃣ 精益商业计划

**提示词：**
```
为这个想法创建简单的精益初创计划：
用 AI 工具帮创作者批量制作视频解读内容，发布到 B 站/小红书

包括目标客户、核心价值主张、营收模式、关键分销渠道，
以及以零或极低成本启动最小可行产品的最快方式。
```

**预期输出：**
- 目标客户：25-35 岁，想学习但没时间
- 价值主张：3 分钟看懂 30 分钟视频
- 营收：平台分成 + 广告 + 付费订阅
- 渠道：B 站、小红书、公众号
- MVP：1 个视频，测试播放量

---

## 5️⃣ MVP 设计

**提示词：**
```
帮我为这个商业想法设计简单的最小可行产品（MVP）：
用 AI 工具制作"YouTube 视频解读"发布到 B 站

解释基础版本应包括什么、如何使用免费工具构建、
以及如何快速推出来测试客户是否愿意付费。
```

**预期输出：**
- MVP 内容：1 个视频，5-8 分钟
- 工具：Whisper（转录）+ 剪映（剪辑）+ TTS（配音）
- 成本：$0（用免费工具）
- 测试：发布后看播放量、点赞、评论

---

## 6️⃣ 获客计划

**提示词：**
```
像增长策略师一样思考。针对这个初创想法：
AI 视频解读 B 站账号

创建分步计划来获得前 50-100 个粉丝，不花广告费。
重点关注社区、社交媒体、合作和直接外联等有机策略。
```

**预期输出：**
1. 第 1 周：发 3 个视频，测试哪种题材好
2. 第 2 周：在相关 subreddit/豆瓣小组分享
3. 第 3 周：和其他 UP 主互推
4. 第 4 周：优化标题和封面，提高点击率

---

## 7️⃣ 变现策略

**提示词：**
```
针对这个初创想法：AI 视频解读 B 站账号

设计可以快速产生营收的简单变现策略。
建议定价选项、早期优惠，
以及将早期用户或兴趣转化为付费客户的实际方式。
```

**预期输出：**
- 平台分成：B 站创作激励（约¥10-30/千播放）
- 广告：接软广（¥500-2000/条）
- 付费订阅：知识星球/爱发电（¥9.9/月）
- 定制服务：帮人做视频解读（¥100-500/个）
