# 🚀 创业启动提示词模板

**来源：** 币圈老王 (@crypto_oldwang)  
**收录时间：** 2026-03-14  
**用途：** 用 AI 辅助创业想法验证、市场研究、MVP 设计

---

## 1️⃣ 初创想法验证器

**用途：** 评估商业想法的可行性

```
像经验丰富的创业导师一样分析。评估这个商业想法：[描述你的想法]。
评估市场需求、目标受众和竞争对手。
突出最大风险、弱点和机会，并建议实际改进方式。
```

**使用场景：**
- 有新想法但不确定是否值得做
- 需要第三方客观评估
- 想提前发现潜在风险

---

## 2️⃣ 高利润细分市场寻找器

**用途：** 基于自身资源找创业方向

```
基于我拥有的技能、兴趣或资源：[描述它们]，
生成 10 个可以用很少或无成本启动的现实初创想法。
重点关注市场需求强、目标客户清晰、快速产生收入的想法。
```

**使用场景：**
- 想创业但不知道做什么
- 预算有限，需要低成本启动
- 想发挥自己的优势

---

## 3️⃣ 客户痛点研究员

**用途：** 深度理解目标客户问题

```
像市场研究员一样思考。针对这个商业想法：[描述想法]，
找出目标客户面临的主要问题或沮丧。
解释为什么这些问题存在、客户如何尝试解决，
以及愿意为什么样的解决方案付费。
```

**使用场景：**
- 验证问题是否真实存在
- 理解客户付费意愿
- 发现未被满足的需求

---

## 4️⃣ 精益初创商业计划构建器

**用途：** 快速创建简易商业计划

```
为这个想法创建简单的精益初创计划：[描述想法]。
包括目标客户、核心价值主张、营收模式、关键分销渠道，
以及以零或极低成本启动最小可行产品的最快方式。
```

**使用场景：**
- 需要快速梳理商业模式
- 准备和合伙人/投资人沟通
- 明确下一步行动

---

## 5️⃣ 最小可行产品创建器

**用途：** 设计 MVP，快速测试市场

```
帮我为这个商业想法设计简单的最小可行产品（MVP）：[描述想法]。
解释基础版本应包括什么、如何使用免费工具构建、
以及如何快速推出来测试客户是否愿意付费。
```

**使用场景：**
- 不知道 MVP 应该包含什么功能
- 预算有限，需要用免费工具
- 想快速上线测试市场反应

---

## 6️⃣ 首批客户获取计划

**用途：** 零预算获取前 100 个客户

```
像增长策略师一样思考。针对这个初创想法：[描述想法]，
创建分步计划来获得前 50-100 个客户，不花广告费。
重点关注社区、社交媒体、合作和直接外联等有机策略。
```

**使用场景：**
- 没有广告预算
- 不知道从哪里找第一批用户
- 需要可执行的获客步骤

---

## 7️⃣ 快速营收策略

**用途：** 设计变现方案，尽快产生现金流

```
针对这个初创想法：[描述想法]，
设计可以快速产生营收的简单变现策略。
建议定价选项、早期优惠，
以及将早期用户或兴趣转化为付费客户的实际方式。
```

**使用场景：**
- 需要尽快产生现金流
- 不确定如何定价
- 想设计有效的转化漏斗

## 📋 使用建议

### 最佳使用顺序

```
1 → 3 → 2 → 4 → 5 → 6 → 7
想法验证 → 痛点研究 → 市场寻找 → 商业计划 → MVP → 获客 → 变现
```

### 提示词优化技巧

1. **具体描述** - 越详细，回答越精准
   - ❌ "我想做个 SaaS"
   - ✅ "我想做一个帮助自由职业者追踪项目时间的 SaaS，目标用户是设计师和开发者"

2. **提供背景** - 包括你的技能、资源、限制
   - "我有 3 年前端开发经验，预算$5000，每周可用 20 小时"

3. **追问细节** - 对模糊建议继续追问
   - "你说的'社交媒体营销'具体指哪个平台？怎么做？"

4. **要求量化** - 让 AI 给出具体数字
   - "预计需要多少时间？""转化率大概多少？""定价区间是多少？"

## 💡 进阶用法

### 组合使用示例

**场景：** 想做一个 AI 写作助手

```
1. 先用提示词 1 验证想法
   → AI 分析：市场需求存在，但竞争激烈

2. 用提示词 3 研究痛点
   → 发现：现有工具太贵，小团队用不起

3. 用提示词 2 找细分市场
   → 建议：专注"非英语母语者写英文邮件"

4. 用提示词 5 设计 MVP
   → 建议：用 GPT API + 简单网页，$0 启动

5. 用提示词 6 获客
   → 建议：在 ESL 论坛、LinkedIn 群组推广

6. 用提示词 7 变现
   → 建议：$9/月，前 100 用户 50% 折扣
```

---

## ⚠️ 注意事项

1. **AI 建议需要验证** - 不要盲目相信，做自己的调研
2. **市场在变化** - AI 训练数据有截止日期，最新趋势需要自己查
3. **执行比想法重要** - 再好的建议，不执行也没用
4. **结合自身情况** - AI 不了解你的全部背景，需要自己调整
