# 📈 短线选股策略 - 使用指南

## ⚠️ 网络问题说明

AkShare 依赖访问国内财经网站（东方财富、深交所等），在中国大陆可能遇到：
- 连接超时
- 数据加载失败
- API 限流

## ✅ 解决方案

### 方案 1：使用代理/特殊网络环境
```bash
# 设置代理后运行
export HTTP_PROXY="http://127.0.0.1:7890"
export HTTPS_PROXY="http://127.0.0.1:7890"
python3 short_term_strategy.py
```

### 方案 2：使用本地数据源（推荐）

安装 Tushare（需要注册获取 token）：
```bash
pip install tushare
```

修改脚本使用 Tushare 数据源。

### 方案 3：手动选股 + 我帮你分析

1. 打开你的炒股软件（同花顺/东方财富等）
2. 使用条件选股功能，设置：
   - 量比 > 1.5
   - 成交额 > 1 亿
   - 涨幅 2%-5%
   - 股价 > 5 日线
3. 把筛选结果告诉我，我帮你进一步分析

## 📊 短线选股条件（供参考）

| 条件 | 参数 | 说明 |
|------|------|------|
| 量比 | > 1.5 | 成交量放大 |
| 成交额 | > 1 亿 | 流动性好 |
| 涨跌幅 | 2%-5% | 强势但不追高 |
| 均线 | 股价>5 日>10 日 | 多头排列 |
| MACD | 金叉或红柱放大 | 趋势向好 |
| 资金流 | 主力净流入 | 有资金关注 |

## 💰 资金管理

```
总资金：10,000 元/天
分散：2-3 只股票
单只：3,000-5,000 元
止损：-2% 到 -3%
止盈：+3% 到 +5%
```

## 📱 每日流程

1. **早盘前**（9:15-9:25）：查看集合竞价，找强势股
2. **开盘后**（9:30-10:00）：观察资金流向，不急于买入
3. **盘中**（10:00-14:30）：等待合适买点
4. **尾盘**（14:30-15:00）：决定是否持仓过夜

## ⚠️ 风险提示

- 短线交易风险高，需要严格执行止损
- 数据仅供参考，不构成投资建议
- 市场有风险，投资需谨慎
