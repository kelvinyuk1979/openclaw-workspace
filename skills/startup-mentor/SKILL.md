# Startup Mentor - 创业导师技能

**版本：** 2.0.0  
**作者：** 用户定制  
**用途：** 创业想法验证、商业计划、MVP 构建、客户获取、营收策略、OpenClaw 变现

---

## 📌 技能描述

本技能提供 8 个核心工具，帮助用户从创业想法到实现营收的全流程指导：

1. **初创想法验证器** - 评估商业想法的可行性
2. **高利润细分市场寻找器** - 基于个人技能生成创业想法
3. **客户痛点研究员** - 深度分析目标客户问题
4. **精益初创商业计划构建器** - 创建简单商业计划
5. **最小可行产品创建器** - 设计 MVP 方案
6. **首批客户获取计划** - 零成本获客策略
7. **快速营收策略** - 快速变现方法
8. **OpenClaw 变现指南** - 6 种已验证的 AI Agent 变现路径（新增）

## 🛠️ 工具列表

### 1. 初创想法验证器

**用途：** 像经验丰富的创业导师一样分析商业想法

**提示词模板：**
```
像经验丰富的创业导师一样分析。评估这个商业想法：[描述你的想法]。

请评估：
1. 市场需求 - 这个需求真实存在吗？规模多大？
2. 目标受众 - 谁会为此付费？用户画像是什么？
3. 竞争对手 - 现有解决方案有哪些？你的差异化在哪里？
4. 最大风险 - 可能导致失败的 3 个最大风险是什么？
5. 弱点分析 - 当前想法的致命弱点是什么？
6. 机会点 - 有哪些被忽视的机会？
7. 实际改进建议 - 具体可执行的改进方案

请用结构化格式输出，突出关键洞察。
```

---

### 2. 高利润细分市场寻找器

**用途：** 基于个人技能/资源生成低成本创业想法

**提示词模板：**
```
基于我拥有的技能、兴趣或资源：[描述它们]，生成 10 个可以用很少或无成本启动的现实初创想法。

每个想法必须满足：
- ✅ 启动成本：低于 $100 或零成本
- ✅ 市场需求：有明确的目标客户愿意付费
- ✅ 收入潜力：能够在 30 天内产生第一笔收入
- ✅ 可执行性：一个人可以启动和运营

对每个想法，请说明：
1. 具体做什么
2. 目标客户是谁
3. 如何获取客户
4. 如何收费/变现
5. 启动步骤（前 7 天做什么）

按收入潜力和可行性排序。
```

---

### 3. 客户痛点研究员

**用途：** 深度分析目标客户面临的问题和付费意愿

**提示词模板：**
```
像市场研究员一样思考。针对这个商业想法：[描述想法]，找出目标客户面临的主要问题或沮丧。

请分析：
1. 核心问题 - 客户最痛的 3 个问题是什么？
2. 问题根源 - 为什么这些问题存在？是市场失灵还是技术限制？
3. 现有解决方案 - 客户目前如何尝试解决？为什么不满意？
4. 付费意愿 - 客户愿意为什么样的解决方案付费？价格区间？
5. 决策因素 - 客户选择解决方案时最看重什么？
6. 购买障碍 - 什么会阻止客户购买？

请提供具体的客户引言/场景示例，让问题更生动。
```

---

### 4. 精益初创商业计划构建器

**用途：** 创建简单的精益创业计划

**提示词模板：**
```
为这个想法创建简单的精益初创计划：[描述想法]。

包括以下要素：
1. 目标客户 - 具体画像（年龄、职业、痛点、消费习惯）
2. 核心价值主张 - 用一句话说清楚你解决什么问题
3. 营收模式 - 如何赚钱？定价策略？
4. 关键分销渠道 - 如何触达客户？（线上/线下）
5. 成本结构 - 主要成本项有哪些？
6. MVP 方案 - 最小可行产品是什么？如何用零/极低成本构建？
7. 关键指标 - 如何衡量成功？（如：转化率、留存率、LTV）
8. 时间表 - 从 0 到上线的关键节点

用表格形式输出，保持简洁可执行。
```

### 5. 最小可行产品创建器

**用途：** 设计 MVP 方案，快速测试市场

**提示词模板：**
```
帮我为这个商业想法设计简单的最小可行产品（MVP）：[描述想法]。

请回答：
1. MVP 核心功能 - 基础版本必须包括什么？（不超过 3 个功能）
2. 免费工具推荐 - 如何使用免费/低成本工具构建？
   - 网站/落地页：
   - 支付系统：
   - 客户沟通：
   - 产品交付：
3. 快速上线方案 - 最快多久能推出？具体步骤？
4. 测试方法 - 如何测试客户是否愿意付费？
5. 成功标准 - 什么信号表明应该继续/放弃？
6. 迭代路径 - 根据反馈如何迭代？

目标是：用最低成本、最快速度验证核心假设。
```

### 6. 首批客户获取计划

**用途：** 零成本获取前 50-100 个客户

**提示词模板：**
```
像增长策略师一样思考。针对这个初创想法：[描述想法]，创建分步计划来获得前 50-100 个客户，不花广告费。

请提供：
1. 目标客户聚集地 - 他们在哪里？（具体社区、平台、群组）
2. 内容营销策略 - 什么内容能吸引他们？
3. 社交媒体策略 - 如何在 Twitter/小红书/抖音获客？
4. 合作机会 - 可以和谁合作互推？
5. 直接外联 - 如何冷启动联系潜在客户？（邮件/私信模板）
6. 社区建设 - 如何建立自己的用户社区？
7. 推荐机制 - 如何让现有用户带来新用户？

按优先级排序，给出具体可执行的步骤和时间表（如：第 1 周做什么，第 2 周做什么）。
```

### 7. 快速营收策略

**用途：** 设计快速变现策略

**提示词模板：**
```
针对这个初创想法：[描述想法]，设计可以快速产生营收的简单变现策略。

请包括：
1. 定价选项 - 建议的价格点和套餐设计
2. 早期优惠 - 如何设计限时优惠吸引首批用户？
3. 转化策略 - 如何将兴趣用户转化为付费客户？
4.  upsell 机会 - 如何增加客单价？
5. 收款方式 - 推荐哪些收款工具？（国内/国际）
6. 营收目标 -  realistic 的第 1 个月/第 3 个月营收目标

重点关注：
- 如何降低首次购买门槛
- 如何建立信任促成交易
- 如何设计自动续费/复购机制

给出具体话术和页面文案建议。
```

### 8. OpenClaw 变现指南 🆕

**背景：** OpenClaw 是一个开源 AI Agent，可以 24 小时持续运行，接入 WhatsApp、Telegram、Discord 等工具，替你执行真实任务。不到 3 周在 GitHub 上获得 263,000+ Stars。

| 1️⃣ | AI 自由职业服务 | $30-500/项目 | ⭐⭐ | 有专业技能者 |
| 2️⃣ | 在 ClawHub 上卖 Skills | $5-100/Skill | ⭐⭐ | 开发者、自动化专家 |

#### 路径 1：AI 自由职业服务

**做什么：** 在 Upwork、Fiverr 等平台提供服务，OpenClaw 执行具体工作

**服务类型：**
- 写作（文章、文案、邮件）
- 编程（代码生成、调试、部署）
- 研究分析（市场调研、竞品分析）
- 自动化处理（数据整理、报告生成）

**操作方式：**
```
你负责：对接客户、确认需求、把控质量
OpenClaw 负责：执行具体工作
```

**启动步骤：**
1. 在 Upwork/Fiverr 创建服务页面
2. 用 OpenClaw 完成 5-6 个项目，让它学习你的标准
3. 逐渐减少介入，让 Agent 独立交付

**提示词：**
```
"我想在 Upwork 提供 [服务类型] 服务。帮我：
1. 设计服务描述和定价
2. 列出 OpenClaw 可以自动化的具体任务
3. 制定质量把控流程
4. 设计客户沟通模板"
```

## ⚠️ 避坑指南

**不稳定的方向（谨慎）：**
- ❌ 炒币/量化/自动交易
- ❌ "睡后收入""全自动被动赚钱"

**原因：**
- 不稳定
- 很快会被套利掉
- 很难长期持续

**稳定的方向（推荐）：**
- ✅ 自动化企业本来就愿意付钱的工作
- ✅ 解决真实痛点，创造真实价值

## 📋 第一周行动计划

| 日期 | 任务 |
|------|------|
| **Day 1** | 安装 OpenClaw，接入 Telegram，告诉它你的基本信息（名字、工作、风格、目标） |

| **Day 4-5** | 从 6 种变现路径中选 1 个最符合你能力和资源的方向 |
| **Day 6-7** | 拿第一个真实客户，或挂出第一个商品/服务页面 |

**核心原则：** 一个不完美的真实客户，比准备两周更有价值

## ⚠️ 注意事项

1. **具体描述** - 想法描述越具体，建议越有针对性
2. **诚实评估** - 不要回避风险和弱点，早期发现问题是好事
3. **快速行动** - 这些工具的目的是帮助你快速验证，不是完美计划
4. **客户验证** - 所有建议都需要实际客户验证，不要只停留在纸面
5. **迭代思维** - 根据反馈持续调整方向和策略
