# TradingView 量化投资分析专家

基于 TradingView MCP 工具的专业量化投资分析系统，提供智能选股、技术形态识别、市场复盘、风险管理、事件驱动分析等功能。

## 核心功能

### 1. 智能选股
- 多因子综合评分
- 技术面 + 基本面双重筛选
- 支持 A 股、美股、港股、加密货币

### 2. 技术形态识别
- 自动识别经典形态（双底、头肩顶、三角形等）
- 提供置信度评估和交易策略
- 成功率统计

### 3. 市场复盘
- 追踪热点板块
- 资金流向分析
- 发现投资机会

### 4. 风险管理
- 仓位建议（Kelly 公式）
- 止损止盈位计算
- 波动率分析
- 投资组合优化

### 5. 事件驱动分析
- 财报日历追踪
- 政策新闻分析
- 经济数据发布

### 6. 基本面筛选
- 高股息股票筛选
- 低估值股票筛选
- 盈利能力分析

### 7. 板块轮动
- 识别强势板块
- 轮动趋势分析
- 多板块对比

## 工作流程

### 核心分析
- **深度个股分析** - 报价 + 多时间框架价格 + 技术指标 + 新闻 + 日历
- **智能选股** -  leaderboard 多列筛选 + 技术分析
- **基本面筛选** - 估值/盈利能力/股息率筛选
- **形态识别** - 价格数据 + 技术指标 + 形态库
- **多时间框架分析** - 日/周/月线趋势确认

### 市场与板块
- **市场复盘** - 涨跌榜 + 新闻
- **板块轮动** - 表现列 + 多板块对比
- **新闻简报** - 多国多语言财经新闻

### 风险与事件
- **风险评估** - 历史数据 + 波动率计算
- **事件分析** - 日历 + 新闻 + 搜索
- **日历追踪** - 经济/财报/营收/IPO 事件

**用户：** "我有 10 万资金，想买普元信息，应该买多少？"

```
账户：¥100,000

当前股价：¥25
止损位：¥23.75（-5%）

计算：
- 最大风险：¥2,000（2%）
- 止损距离：¥1.25（5%）
- 仓位：¥2,000 ÷ 0.05 = ¥40,000

建议：
- 买入金额：¥40,000
- 股数：1,600 股
- 止损价：¥23.75
- 目标价：¥30（+20%）
- 风险收益比：1:4

⚠️ 单只股票不超过总资金的 20%
```

## 核心规则

### 元数据优先原则

**调用 leaderboard 前必须先获取元数据：**

1. `type='markets'` → 获取 `market_code`
2. `type='tabs'` + `asset_type` → 获取 `tab` 值
3. `type='columnsets'` → 获取 `columnset` 值

### 工具选择速查

| 需求 | 工具 | 关键参数 |
|------|------|---------|
| 搜索标的 | `search_market` | query, filter |
| 实时报价 | `get_quote` | symbol, session |
| K 线数据 | `get_price` | timeframe(1/5/15/30/60/240/D/W/M) |
| 技术分析 | `get_ta` | include_indicators=true |
| 排行榜 | `get_leaderboard` | columnset(overview/performance/valuation...) |
| 新闻 | `get_news` | market_country, lang |
| 财经日历 | `get_calendar` | type(economic/earnings/revenue/ipo) |

## 风险管理提示

⚠️ **重要：**
- 所有分析仅供参考，不构成投资建议
- 投资有风险，入市需谨慎
- 建议设置止损（通常 5-8%）
- 单只股票不超过总资金的 20%
- 先用模拟账户练习
