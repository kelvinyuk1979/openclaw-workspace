# 📊 TradingView 量化投资分析

基于 TradingView 数据的专业量化分析技能

## 快速开始

### 1. 配置 MCP Server

**步骤：**
1. 访问 https://rapidapi.com/hypier/api/tradingview-data1
2. 注册并获取免费 API Key
3. 添加到 MCP 配置文件
4. 重启 Cursor/Claude Desktop

### 2. 开始使用

直接用自然语言提问：

```
- "帮我从 A 股科技板块筛选强势股"
- "深度分析普元信息（688118）"
- "今天 A 股市场怎么样？"
- "我有 10 万资金，应该买多少？"
```

## 核心功能

| 功能 | 说明 |
|------|------|
| 🎯 智能选股 | 多因子评分，技术面 + 基本面筛选 |
| 📐 形态识别 | 自动识别双底、头肩顶等经典形态 |
| 📈 市场复盘 | 热点板块、资金流向分析 |
| 🛡️ 风险管理 | 仓位建议、止损止盈计算 |
| 📰 事件驱动 | 财报日历、新闻分析 |
| 💰 基本面筛选 | 高股息、低估值、盈利能力 |
| 🔄 板块轮动 | 识别强势板块和轮动趋势 |

## 使用场景

### 场景 1：选股

```
帮我筛选 A 股科技股：
- RSI 30-70，MACD 金叉
- PE < 30，ROE > 15%
- 市值 > 50 亿
```

### 场景 2：个股分析

```
深度分析 XXXX（股票代码）：
- 多时间框架技术分析
- 基本面数据
- 近期新闻
- 买卖建议
```

### 场景 3：仓位管理

```
我有 10 万资金，想买 XXXX：
- 应该买多少？
- 止损位设在哪？
- 风险收益比如何？
```

### 场景 4：市场复盘

```
今天 A 股/美股市场怎么样？
- 热点板块
- 投资机会
- 风险提示
```

## 风险提示

⚠️ **此技能仅用于教育目的**

- 不构成投资建议
- 投资有风险，入市需谨慎
- 建议设置止损（5-8%）
