# 📈 量化自动交易系统

多策略投票的自动化交易系统 - 4 策略共识 + 自动风控

## 快速开始

### 1. 安装依赖

```bash
pip install numpy requests
```

### 2. 运行系统

```bash
# 查看状态
python3 trading_system.py status

# 启动自动交易
python3 trading_system.py run
```

## 核心特性

| 特性 | 说明 |
|------|------|
| 🗳️ **多策略投票** | 4 种策略共识决策，减少误判 |
| 💰 **自动仓位** | 每笔最大 10% 资金 |
| 🛑 **止损止盈** | -5% 止损，+10% 止盈 |
| 📊 **实时数据** | 连接 Hyperliquid API |
| 🧪 **模拟交易** | 虚拟$10,000 起步 |

## 4 大策略

| 策略 | 原理 | 信号 |
|------|------|------|
| 动量 | 追涨杀跌 | RSI<40 多，RSI>60 空 |
| 均值回归 | 超买超卖反转 | RSI<30 多，RSI>70 空 |
| MACD 交叉 | 趋势跟踪 | MACD>0 多，MACD<0 空 |
| 超级趋势 | 趋势跟随 | 价格>MA20 多，价格<MA20 空 |

## 支持币种

- BTC (比特币)
- ETH (以太坊)
- SOL (Solana)
- XRP (瑞波币)

## 风险控制

```
止损：-5%
止盈：+10%
风险收益比：1:2
单笔最大：10%
最大持仓：4 个币种
```

## 交易流程

```
1. 检查现有持仓 → 触发止损/止盈？→ 平仓
2. 获取市场数据 → 价格、RSI、MACD
3. 4 策略投票 → 多数决定信号
4. 执行交易 → 开仓/持仓/观望
5. 等待下一轮（5 分钟后）
```
