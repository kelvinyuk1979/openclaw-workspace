# 📈 量化自动交易系统

基于多策略投票的自动化交易系统，提供共识决策、自动仓位管理、止损止盈等功能。

## 系统架构

```
量化交易系统
├── 多策略投票（4 种策略）
├── 自动仓位管理
├── 风险控制（止损/止盈）
└── 模拟交易模式
```

## 核心功能

### 1. 多策略投票系统

**4 种策略共识决策：**

| 策略 | 原理 | 信号条件 |
|------|------|---------|
| **动量策略** (momentum) | 追涨杀跌 | RSI<40 做多，RSI>60 做空 |
| **均值回归** (mean_reversion) | 超买超卖反转 | RSI<30 做多，RSI>70 做空 |
| **MACD 交叉** (macd_cross) | 趋势跟踪 | MACD>0 做多，MACD<0 做空 |
| **超级趋势** (supertrend) | 趋势跟随 | 价格>MA20 做多，价格<MA20 做空 |

**投票机制：**
- 每个策略一票（LONG/SHORT/HOLD）
- 多数票决定最终信号
- 平票时保持观望（HOLD）

### 2. 自动仓位管理

```
仓位计算：
- 最大仓位：总资金的 10% 每笔
- 账户资金：$10,000（模拟）
- 单笔最大：$1,000
- 最多持仓：4 个币种
```

### 3. 风险控制

| 参数 | 值 | 说明 |
|------|-----|------|
| 止损 (SL) | -5% | 亏损 5% 自动平仓 |
| 止盈 (TP) | +10% | 盈利 10% 自动平仓 |
| 风险收益比 | 1:2 | 亏 5% 赚 10% |

### 4. 支持币种

- **BTC** (比特币)
- **ETH** (以太坊)
- **SOL** (Solana)
- **XRP** (瑞波币)

## 策略详细说明

### 1. 动量策略 (Momentum)

**原理：** 强者恒强，弱者恒弱

**信号：**
- LONG: RSI < 40（超卖，可能反弹）
- SHORT: RSI > 60（超买，可能回调）
- HOLD: RSI 40-60（震荡）

**适用场景：** 趋势明显的市场

### 2. 均值回归 (Mean Reversion)

**原理：** 价格会回归平均值

**信号：**
- LONG: RSI < 30（极度超卖）
- SHORT: RSI > 70（极度超买）
- HOLD: RSI 30-70（正常）

**适用场景：** 震荡市场

### 3. MACD 交叉 (MACD Cross)

**原理：** 快慢线交叉表示趋势变化

**信号：**
- LONG: MACD 柱状图 > 0（多头动能）
- SHORT: MACD 柱状图 < 0（空头动能）
- HOLD: MACD ≈ 0（无明确方向）

**适用场景：** 趋势启动初期

### 4. 超级趋势 (Supertrend)

**原理：** 基于移动平均的趋势跟踪

**信号：**
- LONG: 价格 > MA20（上升趋势）
- SHORT: 价格 < MA20（下降趋势）

**适用场景：** 明确趋势市场

## 交易流程图

```
开始
  ↓
检查现有持仓
  ↓
是否有持仓触及止损/止盈？
  ├─ 是 → 平仓 → 更新余额
  └─ 否 → 继续
  ↓
遍历每个币种
  ↓
获取市场数据（价格、RSI、MACD）
  ↓
4 个策略分别投票
  ↓
统计投票结果
  ├─ 多数做多 → 开多单（如无持仓）
  ├─ 多数做空 → 开空单（如无持仓）
  └─ 平票 → HOLD
  ↓
等待下一轮（每 5 分钟）
```

## 风险管理

### 仓位管理

```
总资金：$10,000
单笔最大：10% = $1,000
最大持仓：4 个币种
最大风险敞口：40% = $4,000
```

### 止损策略

| 类型 | 触发条件 | 操作 |
|------|---------|------|
| 固定止损 | 亏损 5% | 自动平仓 |
| 固定止盈 | 盈利 10% | 自动平仓 |
| 时间止损 | 持仓>24 小时无盈利 | 可选平仓 |

### 风险收益比

```
止损：-5%
止盈：+10%
风险收益比：1:2

即：冒 1 块钱风险，赚 2 块钱
胜率只需>33% 即可盈利
```

## 模拟 vs 实盘

| 特性 | 模拟模式 | 实盘模式 |
|------|---------|---------|
| 资金 | 虚拟$10,000 | 真实资金 |
| 数据 | 真实市场数据 | 真实市场数据 |
| 交易 | 模拟成交 | API 下单 |
| 风险 | 无风险 | 有亏损风险 |

**⚠️ 重要提示：**
- 先用模拟模式测试至少 1 个月
- 确认策略有效后再考虑实盘
- 实盘只用输得起的钱
