## 💡 使用方式

### 提示词 1：交易想法生成器

**用途：** 扫描市场，生成高概率交易设置

**模板：**
```
扫描今日市场，为 [插入股票/指数/板块] 生成 5 个高概率交易设置。
包括入市价、出市目标、止损和风险回报比。
基于技术和基本面因素解释每个设置的有效性。
```

**示例：**
```
扫描今日加密货币市场，为 BTC/ETH/SOL 生成 5 个高概率交易设置。
包括入市价、出市目标、止损和风险回报比。
```

**输出格式：**
| 设置 | 方向 | 入市价 | 目标价 | 止损 | R:R | 有效性 |
|------|------|--------|--------|------|-----|--------|

---

### 提示词 2：自动技术分析师

**用途：** 深度技术分析，生成交易信号

**模板：**
```
使用日线和周线图分析 [插入股票/代码]。
分解支撑/压力位、趋势线、移动平均线和动量指标。
提供带有理由的逐步交易信号（买入/持有/卖出）。
```

**示例：**
```
使用日线和周线图分析 BTC。
分解支撑/压力位、趋势线、MA20/50/200、RSI、MACD。
提供带有理由的逐步交易信号。
```

**输出格式：**
- 支撑位：S1, S2, S3
- 压力位：R1, R2, R3
- 趋势：日线/周线方向
- 指标：RSI, MACD, MA 状态
- 信号：买入/持有/卖出 + 理由

---

### 提示词 3：新闻转换交易

**用途：** 将新闻转化为可执行的交易洞察

**模板：**
```
总结 [插入公司/板块] 的最新新闻，并将其转化为交易影响。
提供可能的短期和长期影响，预期价格变动范围和推荐持仓。
```

**示例：**
```
总结美联储最新利率决议新闻，转化为加密货币交易影响。
提供短期（1 周）和长期（1 月）影响，预期 BTC 价格变动范围。
```

**输出格式：**
- 新闻摘要
- 短期影响（1-7 天）
- 长期影响（1-4 周）
- 预期价格范围
- 推荐持仓方向

---

### 提示词 4：策略回测

**用途：** 回测交易策略，验证有效性

**模板：**
```
对 [插入股票/指数] 的 [插入交易策略] 进行最近 [插入时间段] 的回测。
展示胜率、利润因子、最大回撤和提高优势的改进方法。
```

**示例：**
```
对 BTC 的"RSI<30 买入，RSI>70 卖出"策略进行最近 6 个月的回测。
展示胜率、利润因子、最大回撤和改进方法。
```

**输出格式：**
| 指标 | 数值 |
|------|------|
| 交易次数 | N |
| 胜率 | % |
| 利润因子 | X |
| 最大回撤 | % |
| 夏普比率 | X |
| 改进建议 | ... |

---

### 提示词 5：投资组合风险管理

**用途：** 分析风险暴露，提供对冲建议

**模板：**
```
分析我的投资组合：[插入代码和%分配]。
突出弱点、过度暴露和隐藏相关性。
建议风险调整再平衡和对冲策略，以防 20% 市场下跌。
```

**示例：**
```
分析我的投资组合：BTC 40%, ETH 30%, SOL 20%, XRP 10%。
突出弱点、过度暴露和相关性。
建议对冲策略以防 20% 市场下跌。
```

**输出格式：**
- 当前配置
- 风险暴露分析
- 相关性矩阵
- 弱点识别
- 再平衡建议
- 对冲策略（防 20% 下跌）

---

### 提示词 6：交易日志分析

**用途：** 分析交易行为，识别改进点

**模板：**
```
回顾我的最近 20 笔交易：[插入带入市/出市和结果的交易]。
识别重复性错误、错过的机会和行为偏见。
给我 3 个可立即提高一致性的个性化规则。
```

**示例：**
```
回顾我的最近 20 笔加密货币交易：
[列出交易记录]
识别重复性错误和行为偏见。
给我 3 个提高一致性的规则。
```

**输出格式：**
- 交易统计（胜率、平均盈亏）
- 重复性错误 Top 3
- 错过的机会
- 行为偏见识别
- 3 条个性化改进规则

---

### 提示词 7：全自动交易计划

**用途：** 生成完整的日程化交易计划

**模板：**
```
为 [插入市场/资产] 设计日常交易计划。
包括盘前扫描、开盘策略、午间调整和收盘策略。
以时间戳清单形式呈现，我可以像专业交易员一样遵循。
```

**示例：**
```
为加密货币市场设计日常交易计划。
包括盘前扫描、开盘策略、午间调整和收盘策略。
以时间戳清单形式呈现。
```

**输出格式：**
| 时间 | 任务 | 检查项 |
|------|------|--------|
| 08:00 | 盘前扫描 | 新闻、隔夜走势、关键位 |
| 09:30 | 开盘策略 | 观察开盘方向、成交量 |
| 12:00 | 午间调整 | 上午复盘、调整计划 |
| 16:00 | 收盘策略 | 平仓/持仓决策 |
| 20:00 | 盘后复盘 | 交易日志、明日计划 |
