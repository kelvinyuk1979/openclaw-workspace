# 💹 Day Trading Investor Pro

专业的 AI 交易导师技能 - 技术分析 + 风险管理 + 交易心理

## 快速开始

```bash
# OpenClaw 会自动加载 workspace/skills/ 目录下的技能
# 无需额外安装
```

## 使用示例

### 1. 学习技术指标

```
什么是 RSI 背离？如何交易？
```

### 2. 计算仓位

```
账户$10000，每笔风险 2%，止损距离 5%，应该买多少？
```

### 3. 策略审计

```
我想在突破 20 日均线时买入，这个策略怎么样？
```

### 4. 风险管理

```
如何设置止损位？有哪些止损策略？
```

## 核心功能

| 功能 | 说明 |
|------|------|
| 📊 技术分析 | K 线形态、指标解释、趋势识别 |
| 💰 仓位计算 | 根据风险和止损自动计算 |
| 🔍 策略验证 | 审计交易计划，指出优缺点 |
| 🧠 交易心理 | 纪律建议、情绪管理 |

## 核心原则

1. **风控优先** - 每笔交易必须带止损
2. **教育为主** - 解释原理，不只是给答案
3. **不荐股** - 只教方法，不推荐具体交易

## 适用场景

- ✅ 学习技术分析
- ✅ 验证交易策略
- ✅ 计算风险仓位
- ✅ 理解市场机制
- ❌ 跟单推荐
- ❌ 保证盈利
